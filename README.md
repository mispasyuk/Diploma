# Description
This repository contains files that were used for diploma "Synthetic video recognition based on facial micro-expression analysis for online conferences". 

# Structure:
- Folder 'training_notebooks' contains .ipynb files with training code for explored models.
- Folder 'detector_pipeline' contains files for cloning and using them for deepfake recognition on one video sample.
- Other folders contant code files that were used for collecting and processing videos for private dataset.
